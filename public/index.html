Let your agent swap any token via Jupiter. Non-custodial, we return the transaction, your agent signs locally.

Non-Custodial

We never hold your keys or funds. Your agent signs transactions locally and submits directly to Solana.

Jupiter Routing

Best swap rates across all Solana DEXs via Jupiter aggregator. Any token pair, optimal execution.

Simple Integration

One API call returns a base64 transaction. Works with any language or AI framework.

Get started in seconds

cURL
curl -X POST https://agents.veertx.com/v1/swap \
  -H "Authorization: Bearer vrtx_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "inputToken": "SOL",
    "outputToken": "USDC",
    "amount": "100000000",
    "agentPublicKey": "YourSolanaPublicKey...",
    "slippageBps": 50
  }'
